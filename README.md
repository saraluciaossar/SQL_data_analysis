# SQL Data Analysis - IT Academy
This repository contains SQL exercises and data analysis projects developed during the IT Academy bootcamp, using MySQL.
The goal of this repository is to practice and demonstrate core SQL skills applied to real-world data scenarios, including data exploration, filtering, and aggregation.

### Skills Demostrated
- Data querying and filtering
- Aggregations and grouping
- Working with relational databases
- Writing complex queries using joins and subqueries
- Extracting insights from structured data

### Purpose
This repository is part of my transition into data analysis, where I am strengthening my skills in SQL, Python, and data visualization tools.
